# Demo Playwright

This repository contains automation scripts created that target specific functionalities of the **SauceDemo** and **Restful-Api** testing sites. The scripts are written in **JavaScript** using the  **Playwright** automate testing framework.  One test converted to use BDD (Behavior Driven Development) with **Cucumber**.

## Setup Instructions

 1. Clone repository
 2. Install package dependencies **npm install**
 3. Run tests using commands from package.json
- npm run apitest
- npm run logintest
- npm run end2endtest
- npm run bddtest

## Urls to testing sites used

- https://www.saucedemo.com/
- https://restful-api.dev/
